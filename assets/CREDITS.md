# Asset Credits

The base repository contains original procedural geometry and source code only.

When external scan assets are added, record:

- asset name,
- creator,
- source URL,
- license,
- modifications and reduction ratio.

Example:

```text
Stone Ruin Cluster
Creator: Example Artist
Source: Example URL
License: CC0
Modifications: decimated to 18%, exported as GLB, texture size reduced to 1k.
```
